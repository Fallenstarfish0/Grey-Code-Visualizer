# Gray Code Visualizer

## Description
Interactive visualization of Binary Reflected Gray Code algorithm showing how consecutive codes differ by exactly one bit.

## Features
- Adjustable bit count (2-6 bits)
- Step-by-step animation with speed control
- Algorithm explanation and real-world applications
- Interactive sequence navigation

## Files
- `gray-code-viz.jsx` - Original JSX component
- `Gray Code Visualizer Setup 1.0.0.exe` - Windows executable installer
- `src/` - React source code

## How to Run
Double-click the executable file to launch the application.
